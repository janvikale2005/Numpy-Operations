# NumPy Array Operations

This repository contains a collection of **NumPy operations** on **1D, 2D, and 3D arrays**. It is designed for beginners and intermediate learners to understand array manipulation and basic operations in Python using NumPy.

## Features

- **1D Array Operations**: Creation, indexing, slicing, arithmetic operations, and aggregation functions.
- **2D Array Operations**: Matrix creation, reshaping, transposing, element-wise operations, and more.
- **3D Array Operations**: Advanced multi-dimensional array manipulation.
- **Examples & Explanations**: Each operation is accompanied by a code example and output.

## Installation

1. Make sure you have Python installed (3.x recommended).
2. Install NumPy if not already installed:

```bash
pip install numpy
```
3.Open the Jupyter Notebook file (.ipynb) in Jupyter Notebook or JupyterLab.
